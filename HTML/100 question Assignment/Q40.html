Why Do We Use the <colgroup> Tag in a Table?

The <colgroup> tag is used to group one or more columns in a table so that common styles can be applied to those columns.

Example

<table border="1">

    <colgroup>
        <col style="background-color: lightyellow;">
        <col style="background-color: lightblue;">
    </colgroup>

    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>

    <tr>
        <td>Aayush</td>
        <td>22</td>
    </tr>

</table>
