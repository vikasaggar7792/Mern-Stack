What is the Relationship Between the Border and Rules Attributes?

Both border and rules are attributes used with HTML tables.

  • border specifies the thickness of the table border.
  • rules specifies which internal borders between rows and columns should be displayed.

Example

<table border="1" rules="all">
    ...
</table>

Note: The rules attribute is obsolete in HTML5. CSS is recommended for table borders.
