What are Attributes in HTML?

HTML attributes provide additional information about HTML elements. They are always written inside the opening tag as name-value pairs.

Syntax

<tagname attribute="value">

Example

<a href="https://www.google.com">Google</a>

<img src="image.jpg" alt="Sample Image">

Common Attributes

  • id
  • class
  • href
  • src
  • alt
  • style
  • title
