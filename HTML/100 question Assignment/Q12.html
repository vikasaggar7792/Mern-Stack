What is DNS?

DNS (Domain Name System) is the internet's phonebook. It translates a domain name into an IP address so that browsers can locate and connect to the correct server.

Example

google.com → 142.250.183.14

How DNS Works

  1. User enters a website name.
  2. The browser sends a DNS request.
  3. DNS finds the corresponding IP address.
  4. The browser connects to the web server.
  5. The requested webpage is displayed.
