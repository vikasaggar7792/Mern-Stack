What is Drag and Drop in HTML5?

HTML5 Drag and Drop allows users to drag an element from one location and drop it into another using JavaScript event handling.

Applications

  • File upload systems
  • Kanban boards
  • Image galleries
  • Interactive games

Example

draggable="true"
ondragstart="drag(event)"
ondrop="drop(event)"
ondragover="allowDrop(event)"
