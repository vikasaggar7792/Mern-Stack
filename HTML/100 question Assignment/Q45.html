What is Semantic HTML?

Semantic HTML uses meaningful HTML tags that describe the purpose of the content instead of only defining its appearance.

Semantic elements improve:

  • Code readability
  • Accessibility
  • Search Engine Optimization (SEO)
  • Website structure

Examples of Semantic Tags

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>
