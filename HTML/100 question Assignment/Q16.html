What are Semantic Tags?

Semantic tags are HTML elements that clearly describe the purpose and meaning of their content. They improve code readability, accessibility, and search engine optimization (SEO).

Common Semantic Tags

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>

Example

<header>
    <h1>My Website</h1>
</header>

<main>
    <section>
        Welcome to my website.
    </section>
</main>

<footer>
    Copyright © 2026
</footer>
