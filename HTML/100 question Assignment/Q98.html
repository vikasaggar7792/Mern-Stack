How to Make Radio Buttons and Checkboxes? How to Implement Them?

Example

<form>

Gender:
<input type="radio" name="gender" value="Male"> Male
<input type="radio" name="gender" value="Female"> Female

<br><br>

Skills:
<input type="checkbox" name="skill" value="HTML"> HTML
<input type="checkbox" name="skill" value="CSS"> CSS
<input type="checkbox" name="skill" value="JavaScript"> JavaScript

</form>

Difference

  • Radio buttons allow only one option to be selected.
  • Checkboxes allow multiple options to be selected.
