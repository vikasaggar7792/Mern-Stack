How to Create a Hyperlink in HTML?

A hyperlink is created using the <a> (anchor) tag. It allows users to navigate to another webpage, website, file, or a specific section of the current page.

Syntax

<a href="https://www.google.com">Visit Google</a>

Example

<a href="https://www.prepbytes.com" target="_blank">
    Visit PrepBytes
</a>
