What is DNS and When Does it Come into the Picture?

DNS (Domain Name System) is a system that translates a domain name (such as www.google.com) into an IP address that computers use to communicate.

When Does DNS Come into the Picture?

  1. You enter a website URL in the browser.
  2. The browser sends a DNS request.
  3. DNS returns the corresponding IP address.
  4. The browser connects to the web server using that IP address.
