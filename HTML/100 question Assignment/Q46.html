What is an Image Map?

An Image Map allows different areas of a single image to act as clickable links. It is created using the <map> and <area> tags.

Example

<img src="image.jpg" usemap="#myMap">

<map name="myMap">
    <area shape="rect"
          coords="34,44,270,350"
          href="https://www.google.com">
</map>
