Difference Between HTML5 and HTML4

HTML4 HTML5
No semantic elements. Introduced semantic elements like header, section, article, footer, and nav.
No built-in audio or video support. Supports audio and video using <audio> and <video> tags.
Limited form input types. Provides new input types like email, date, color, and number.
No Canvas support. Supports Canvas and SVG graphics.
Required plugins like Flash. No external plugins are required.
