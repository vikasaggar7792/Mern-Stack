What are the Limits of the Text Field Size?

The size of a text field in HTML can be controlled using different attributes depending on the requirement.

Important Attributes

  • size – Specifies the visible width of the input field.
  • maxlength – Specifies the maximum number of characters that can be entered.
  • minlength – Specifies the minimum number of characters required.

Example

<input type="text"
       size="30"
       maxlength="50"
       minlength="5">
