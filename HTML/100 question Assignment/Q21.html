How Many Types of Headings are There? How to Decide Which to Use?

HTML provides six heading tags:

  • <h1>
  • <h2>
  • <h3>
  • <h4>
  • <h5>
  • <h6>

Usage

  • h1 - Main title of the page (only one is recommended).
  • h2 - Main section headings.
  • h3 - Subheadings.
  • h4-h6 - Smaller sub-sections.
