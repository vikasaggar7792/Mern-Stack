What is SVG?

SVG (Scalable Vector Graphics) is an XML-based image format used to create vector graphics in web pages. SVG images can be scaled to any size without losing quality.

Advantages

  • Scalable without losing quality.
  • Smaller file size for simple graphics.
  • Can be animated using CSS and JavaScript.

Example

<svg width="120" height="120">
    <circle cx="60" cy="60" r="50"
        fill="blue" />
</svg>
