What are Some Common Lists Used When Designing a Web Page?

HTML provides three main types of lists:

  • Ordered List (<ol>) - Displays items in numbered order.
  • Unordered List (<ul>) - Displays items with bullets.
  • Definition List (<dl>) - Used for terms and their descriptions.

Example

<ol>
    <li>HTML</li>
    <li>CSS</li>
</ol>

<ul>
    <li>Java</li>
    <li>Python</li>
</ul>

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>
